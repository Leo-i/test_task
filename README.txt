
Для сборки проекта из среды Vivado запустите файл /src/tcl/create.tcl (проект соберется на одном уровне с папкой src, поэтому 
создайте отдельную папку и распакйте туда)
Он соберет проект и запустит симуляцию 

Результат симуляции можно наблюдать в консоли. Написанный тестбенч дает сигнал для начала теста, в ходе симуляции 
сравниваются отправленные посылки с принятыми и проверяется регистр, хранящий значения сравнений. В итоге
выводится сообщение о прохождении или непрохождении теста.
Также в ходе симуляции специально генерируется неверная посылка для проверки модуля.

Дизайн отправляет поочередно 4 разные посылки, суммарно 31 раз. При получении некорректного ответа поднимает сигнал сброса для модуля FX3.
